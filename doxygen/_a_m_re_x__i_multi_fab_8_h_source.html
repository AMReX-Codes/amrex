12 //
13 // A Collection of IArrayBoxes
14 //
15 // The iMultiFab class is publically derived from the
16 // FabArray<int,IArrayBox> class. It is a collection (stored as an array) of
17 // IArrayBoxes useful for storing integer data on a domain defined by
18 // a union of rectangular regions embedded in a uniform index space. The
19 // iMultiFab class extends the function of the underlying FabArray class just
20 // as the IArrayBox class extends the funtion of BaseFab<int>. Additional
21 // member functions are defined for I/O and simple arithmetic operations on
22 // these aggregate objects.
23 //
24 // This class does NOT provide a copy constructor or assignment operator.
25 //
